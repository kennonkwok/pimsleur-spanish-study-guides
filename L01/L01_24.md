## Grammar Review

### New Concepts

**1. *Gustar* and reverse-subject verbs**
*Gustar* works differently from English "to like." The thing liked is the grammatical subject; the person is expressed as an indirect object pronoun:

- *¿No te gusta el té?* — Don't you like tea? (lit. "Doesn't tea please you?")
- *No me gusta el té.* — I don't like tea. (lit. "Tea doesn't please me.")
- Use *gusta* when the subject is singular; *gustan* when plural.

**2. *Dame* / *Escríbelo* — tú commands with object pronouns**
Affirmative tú commands have the pronoun attached at the end:

- *Dame una taza de té.* — Give me a cup of tea. (*da* command + *me*)
- *Escríbelo.* — Write it down. (*escribe* command + *lo*)
- *Escríbela.* — Write it down. (*escribe* command + *la* — replaces a fem. noun)
- *Escríbalo* — formal Ud. command + *lo*
- When pronouns are attached, accent marks are added to preserve original stress.

**3. *Voy a* + infinitive with attached pronoun**
The pronoun can attach to the infinitive instead of going before *voy*:

- *Voy a escribirla.* — I'm going to write it down. (*la* = *la dirección*)

**4. *Por qué* vs. *porque***
- *¿Por qué crees que estoy enferma?* — Why do you think I'm sick? (question: two words, accent)
- *Porque quieres tomar té.* — Because you want to have tea. (answer: one word, no accent)

**5. *Llamarse* — introducing yourself**
- *Me llamo Fred Jones.* — My name is Fred Jones. (lit. I call myself Fred Jones)
- Formal question: *¿Cómo se llama usted?*; Informal: *¿Cómo te llamas?*

**6. *Solo* — only / alone**
- *Yo solo tomo té cuando estoy enfermo.* — I only drink tea when I'm sick.
- *Solo* as an adjective meaning "alone": *No quiero ir solo.* (Lesson 26)

### Patterns Drilled

| Pattern | Example |

| Ask why with *por qué* | *¿Por qué crees que estoy enferma?* |

| Say you don't remember | *No recuerdo los números.* |

| Ask if near or far | *¿Estás cerca o lejos?* |

### Formal vs. Informal
Lesson 24 drills **both** registers explicitly. Emergency/accident context uses formal (*usted*): *¿Cómo se llama usted? ¿Dónde vive?* The lesson also practices switching: *¿Dónde vives?* (informal).

### Pronunciation Notes
- *Dame*: stress on first syllable — DÁ-me.
- *Escríbelo*: accent on third syllable — es-CRÍ-be-lo.
- *Escríbala*: formal form — es-CRÍ-ba-la.
- *Dirección*: stress on final syllable — di-rec-CIÓN.

## Verb Conjugations

### dar — to give
**Type:** irregular (yo *doy*; other forms mostly regular)

| | Singular | Plural |
|---|---|---|
| 1st | yo: **doy** | nosotros: damos |
| 2nd | tú: das | vosotros: dais |
| 3rd | él/ella/Ud.: da | ellos/ellas/Uds.: dan |

Lesson 24 drills: *dame* (tú affirmative command = *da* + *me*).

---

### gustar — to please / to like
**Type:** regular -ar (but used with reverse-subject construction)

| | Singular | Plural |
|---|---|---|
| 1st | yo: gusto | nosotros: gustamos |
| 2nd | tú: gustas | vosotros: gustáis |
| 3rd | él/ella/Ud.: **gusta** | ellos/ellas/Uds.: **gustan** |

In practice, only 3rd person forms appear: *gusta* (one thing), *gustan* (multiple things). Lesson 24 drills: *¿No te gusta el té?* / *No me gusta el té.*

## Summary
- **dar** — irregular; yo *doy*; tú command *da* (→ *dame*)
- **gustar** — regular -ar; used reverse-subject; only *gusta/gustan* appear in practice
- **recordar** — stem-changing (o→ue)
- **vivir** — regular -ir
- **poder** — stem-changing (o→ue)
- **escribir** — regular -ir
- **tomar** — regular -ar
- **querer** — stem-changing (e→ie); irregular yo *quiero*
- **estar** — irregular; memorize all 6 forms
