## Grammar Review

### New Concepts

**1. *Estar* for location**
This lesson extends the use of *estar* beyond health to express **where something is located**.

- *¿Dónde está el hotel?* — Where is the hotel?
- *Está aquí.* — It's here.
- *Está allí.* — It's over there.

Contrast with *ser*, which is never used for location.

**2. *Aquí* vs. *allí***
Two key location adverbs:
- *aquí* — here (close to the speaker)
- *allí* — there / over there (away from the speaker)

**3. *Encantado / encantada* — gender-matched introduction response**
When introduced, Spanish speakers say *encantado* (men) or *encantada* (women). The word agrees with the **speaker's** gender, not the listener's.

**4. *Castellano* as an alternative for Spanish**
The lesson introduces *castellano* as a synonym for *español*. The audio instructs learners to use *castellano* for the next several lessons to practice it.

**5. *Nos vemos* — casual farewell**
A more informal goodbye than *adiós*: *Nos vemos* (lit. "we see each other" / "see you").

**6. New nationality: *mexicano / mexicana***
Follows the same gender-agreement pattern as *norteamericano/a*.

### Patterns Drilled

| Pattern | Example |
|---|---|
| Ask where something is | *¿Dónde está el hotel Fernández?* |
| Answer with location | *Está aquí.* / *Está allí.* |
| Negate location | *No está aquí.* |
| Introduce yourself formally | *Soy el señor Gómez.* |
| Pleased to meet you | *Encantado.* / *Encantada.* |
| Casual farewell | *Nos vemos.* |
| State nationality | *Soy mexicano.* / *No soy norteamericano.* |
| Use *pero* (but) | *Un poco, pero no muy bien.* |

### Formal vs. Informal
Lesson 4 uses **formal** register (*usted*, titles). No informal *tú* introduced yet.

### Pronunciation Notes
- *Encantado*: four syllables; stress on third: *en-can-TA-do*. The *-d-* between vowels is very soft, almost like English "th" in "the."
- *Castellano*: four syllables; stress on second: *cas-te-LLA-no*. The *ll* sounds like English "y."
- *Allí*: stress on the final syllable: *a-LLÍ*. Accent mark signals this.
- *Aquí*: stress on the final syllable: *a-QUÍ*.

## Verb Conjugations

### estar — to be (temporary states, location)
**Type:** irregular

| | Singular | Plural |
|---|---|---|
| 1st | yo: estoy | nosotros: **estamos** |
| 2nd | tú: estás | vosotros: estáis |
| 3rd | él/ella/Ud.: **está** | ellos/ellas/Uds.: están |

## Summary
- **entender** — stem-changing (e→ie); nosotros/vosotros keep original stem
- **estar** — irregular; used for location and temporary states; lesson drills *está*
- **hablar** — regular -ar
- **saber** — irregular *yo* (*sé*); otherwise regular -er; supplement preview
- **ser** — irregular; memorize all 6 forms
