# Pimsleur Spanish — Study Guides

Vocabulary tables, grammar notes, and verb conjugation references for **Pimsleur Spanish Levels 1–5** (30 lessons each).

These are companion study guides — not a replacement for the audio lessons. Use them alongside the recordings to reinforce what you hear.

## What's included

Each file covers one lesson and has four sections:

| Section | Contents |
|---|---|
| **Vocabulary** | Every key word/phrase drilled in the audio, with English, part of speech, an example sentence, and notes |
| **Supplement Vocabulary** | Additional words from the printed reading booklet not drilled in the audio |
| **Grammar Review** | New grammar concepts introduced, with explanations and example sentences; plus a table of patterns drilled |
| **Verb Conjugations** | Full present-tense tables for each new verb, with verb type (regular, stem-changing, irregular) and notes on which forms were drilled |

## How these were generated

1. Audio downloaded from a personal Audiobookshelf library
2. Transcribed with [Parakeet TDT 0.6B v3](https://huggingface.co/nvidia/parakeet-tdt-0.6b-v3) (MLX port)
3. Study guides generated by Claude (Sonnet 4.6) reading the transcripts directly

The pipeline source is not included here. The generated markdown is the artifact.

## Accuracy notes

- Vocabulary uses infinitive forms, not conjugated forms
- Verb tables are present indicative only (other tenses noted inline where the lesson introduces them)
- Conjugations were verified against standard Spanish grammar rules — regular verb endings, stem-change patterns, and known irregulars are correct
- Transcript quality is high but not perfect; a handful of proper nouns or fast-speech segments may be slightly off
